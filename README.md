This repository documents ADR-0002, an Architecture Decision Record that establishes the baseline governance framework for program stakeholder management and communications planning.

The objective of this ADR is to define how stakeholders are identified, analyzed, engaged, and communicated with throughout the program lifecycle, using formal governance artifacts and version-controlled execution documentation.

This work was completed as part of the IBM Program Management curriculum on Coursera, and is structured to reflect real-world PMO and enterprise governance standards.


---

Scope

ADR-0002 governs:

Program stakeholder identification and classification

Stakeholder interest and influence analysis

Communication cadence, channels, and ownership

Governance controls for post-baseline change


The ADR is supported by execution artifacts maintained in Excel to preserve formatting integrity and usability.
